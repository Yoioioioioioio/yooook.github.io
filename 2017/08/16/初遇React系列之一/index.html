初遇React系列之一

发表于 2017-08-16 | 分类于 React

笔记：

1.react 是一个创建显示组件的东西(组件化开发)

2.jspm包的管理工具，它里面的SystemJS可以加载不同规范的js模块，里面的babel可以编译JavaScript代码，并且可以将JSX代码编译成JavaScript代码

3.props:【
1.这个属性是用来接受父模块给子模块传递的数据的，也可以从主模块多层传递，每层用this.props.date传递
2.在主模块中写法：今天吃了apples<\/Comment>
3.在子模块中写法：<\div className=”text”>{this.props.children}<\/div> //{this.props.children}获取的是标签内容，既”天吃了apples”,
4.这里的this.props就相当于Comment这个元素对象（Element）
5.this.props.data在子模块中的遍历 【
render() {
let commentNodes = this.props.data.map((comment,key) => {//新版本必须传入key值
return (
//属性不能带引号!!!!!!!
{comment.text}

);
});
return(
<\div>

<\/div>

}

4.state:【
1.可以在父模块中添加属性url，
2.安装jquery，创建构造函数，声明属性【
constructor (props) {
super(props);//必须要写
this.state = {data:[]};//接收返回的数据
$.ajax({
url:this.props.url,//拿到父模块传入的地址
dataType:’json’,
cache:false,
success: comment => {
this.setState({data:comment});//设置this.state.data
},
error: (xhr,status,error) => {
console.log(error);
}
});
}
